¿Cuáles son los 4 tipos de datos en Python?

	Strings, los datos que contienen son cadenas de caracteres. Ej: string_ejemplo = 'Este de data type es un string'
	Boolean, los datos solo pueden tener dos valores true o false. Ej: boolean_ejemplo = false
	Float, los datos son números decimales Ej: float_ejemplo = 0.3
	Integer, los datos son números enteros. Ej: int_ejemplo = 5

¿Qué tipo de convención de nomenclatura debemos usar para las variables en Python?
	Hay varias convenciones que se recogen en el manual de python PEP8. La recomendación a la hora de nombrar variables
lo mejor es que el nombre sea lo más descriptivo posible, breve y en letras minúslas utilizando guiones bajos para los espacios
entre las palabras. 

¿Qué es un heredoc de Python?
	Se llaman así a los strings de varias lineas, es decir, que están compuestos por un bloque de texto. El contenido se escribe entre tres comillas
(simples o dobles) de apertura y tres comillas (simples o dobles) de cierre.

¿Qué es la interpolación de cadenas?
La interpolación de cadenas te permite combinar valores de variables dentro de una cadena de caracteres para formar un mensaje completo.
nombre = "Juan"
edad = 10
mensaje_f_string = f"Hola, me llamo {nombre} y tengo {edad} años."
print(mensaje_f_string)

# Usando el método .format()
mensaje_format = "Hola, me llamo {} y tengo {} años.".format(nombre, edad)
print(mensaje_format)

¿Cuándo debemos usar comentarios en Python?

Cuando sea extrictamente necesario ya que el código debe entenderse por si mismo sin necesidad de información adicional.


¿Cuáles son las diferencias entre las aplicaciones monolíticas y de microservicios?

    Monolíticas: Son aplicaciones que están desarrolladas y desplegadas como una única unidad. Toda la funcionalidad está integrada en un único proceso y se ejecuta en un solo entorno.
    Microservicios: Son aplicaciones divididas en pequeños servicios independientes, cada uno con su propia lógica de negocio y bases de datos. Estos servicios se comunican entre sí a través de protocolos de red, como HTTP.
